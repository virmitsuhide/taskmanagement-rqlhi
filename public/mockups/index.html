📝 Catatan untuk Diskusi

  • Visual style mengikuti homepage RQ LHI saat ini: tipografi editorial (Playfair + Lora), aksen gold, layout tenang.
  • Mockup statis — semua data hardcoded untuk preview. Belum ada interaktivitas drag-drop atau koneksi DB.
  • Yang belum di-mockup (bisa ditambah jika perlu): halaman login guru, daftar siswa per halaqoh, halaman admin manage halaqoh, dashboard kepala_rq dengan agregat tahfidz semua divisi.
  • Setelah desain disepakati: kita bikin dokumen skema SQL + roadmap eksekusi, lalu mulai Fase 0 (foundation DB) → Fase 1 (auth guru) → Fase 2 (tahsin/tahfidz harian).
